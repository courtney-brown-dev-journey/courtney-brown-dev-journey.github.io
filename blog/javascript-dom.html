An analogy to describe the differences between HTML and CSS

HTML is the wardrobe and CSS is the clothes. They both exist on their own, but HTML is like a home for CSS and it’s what’s inside a wardrobe/ a structure that is interesting.

Explain control flow and loops using an example process from everyday life

Loops are used if you want to run the same code over and over again, each time with a different value. It’s similar to a habit, e.g. a ritual before a job interview. First you choose an outfit, then you choose another outfit, you do your hair, your make up and then revise your notes. This would repeat each time there’s an interview, like a loop.


Control flow is the order in which the computer runs the script. This is similar to ‘choose-your-own-adventure’ novels, and video games. Certain things may happen if events happen in the right order, e.g. if you choose to go down the left path in the video game, you’ll meet the troll. But if you go down the right path, you will meet the cat who holds the next clue.

Explain what functions are and why they are useful.

A function is a set of statements that performs a task or calculates values. Functions can have inputs that change how the function behaves, so you’re able to personalise content on the page. For functions you need to give them a parameter at the top, which is like a task e.g. greet.

Then at the bottom you have an argument which is really just the options of how the greeting can happen.
